üîß TROUBLESHOOTING

Termux Error:

```bash
# Jika ada error storage:
termux-setup-storage

# Jika error permission:
chmod +x floks_framework.py

# Jika pip error:
pkg install python-pip
```

Linux Error:

```bash
# Jika permission error:
sudo python3 floks_framework.py

# Jika dependency error:
sudo apt install python3-dev libffi-dev libssl-dev -y
```

Windows Error:

```bash
# Jika Python tidak dikenali:
# - Download ulang Python dari python.org
# - Centang "Add Python to PATH"
# - Restart Command Prompt

# Jika pip error:
python -m pip install --upgrade pip
```

macOS Error:

```bash
# Jika brew error:
xcode-select --install

# Jika permission error:
pip3 install --user [package-name]
