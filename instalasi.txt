1. PRE-INSTALASI (WAJIB)

```bash

sudo pacman -S git python python-pip

# macOS
brew install git python3
```

3. DOWNLOAD REPOSITORY

```bash

```

4. MASUK KE DIRECTORY

```bash
cd Floks-cyber
```

5. CEK KONTEN FILE

```bash

```

6. INSTAL REQUIREMENTS PYTHON

```bash
# Jika ada requirements.txt
pip install -r requirements.txt

# Jika tidak ada, install manual
pip install requests bs4 selenium colorama
```

7. JALANKAN TOOLS

```bash
python flox-cyber.py
# ATAU
python3 flox-cyber.py
```

Jika Python tidak dikenali:

```bash
# Cek versi Python
```

Jika pip error:

```bash
# Upgrade pip dulu
pip install --upgrade pip

# Atau gunakan pip3
pip3 install requests bs4 selenium
```

Jika permission denied:

```bash
# Beri permission executable
chmod +x flox-cyber.py

# Coba jalankan lagi
python flox-cyber.py
```
